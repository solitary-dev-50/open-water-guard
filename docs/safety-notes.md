# Safety Notes

## Project Position

Open Water Guard is a DIY controller-board project and an open sensing experiment.
The first release is for sensing and local alerting only.

It is not sold as:

- a finished anti-leak product
- a valve product
- a plumbing-parts product
- an installation business

## What This Project Does Not Claim

This project does not claim:

- certified water-damage protection
- insurance-grade protection
- prevention of all water damage

It is documentation and controller-board experimentation for local water sensing and local alerts.
It is fully offline, uses no cloud path, and does not upload household water data.

## DIY Testing First

Users should begin in a safe desktop water-test environment.

That means:

- bench water loop first
- visible tubing first
- small pump first
- catch tray first
- local alerts first

The project is easier to understand and safer to debug when tested on a bench before any larger physical integration.

## User Responsibility

How a user connects the controller board into a larger personal setup is the user's own decision.

This repository provides:

- controller-board knowledge
- sensing logic ideas
- test-bench references
- documentation for DIY experimentation

It does not provide a promise that every installation method is safe, correct, or appropriate.

## Practical Use Boundary

Treat Open Water Guard like a board-level DIY experiment:

- test it locally
- observe the behavior
- understand the limits
- avoid exaggerated safety claims

That is the right level for the first open detection layer.
