# PCB 设计笔记

这份文档记录 Open Water Guard (OWG) 第一版 PCB 的设计边界和检查项。

PCB 的目标不是把所有功能一次塞满，而是把第一版安全闭环做稳：传感器可靠、阀门动作可靠、断电可见、调试方便、安装不容易接错。

## 设计目标

第一版 PCB 应优先满足：

- ESP32-S3 稳定运行。
- 3-4 路有线漏水探头输入。
- 1 路水流传感器输入。
- 电动球阀驱动接口。
- 蜂鸣器和状态灯。
- 实体按钮。
- 主电源 + 备用电源路径。
- 电池电量或备用电源状态检测。
- 关键接口防呆、保护和清晰丝印。

## 板上功能分区

建议按功能分区布局：

- 主控区：ESP32-S3、晶振、复位、BOOT、下载接口。
- 传感器输入区：漏水探头、水流传感器、水压传感器预留。
- 阀门驱动区：电机驱动、继电器或 MOSFET、反灌保护。
- 电源区：主电源、备用电源、降压、电池检测。
- 人机交互区：按钮、蜂鸣器、状态灯。
- 扩展区：Iris Node/ESP-NOW 相关预留、家庭自动化接口预留。

强电流路径和弱信号输入不要混在一起。阀门驱动区、电源区和传感器输入区应保持清晰边界。

## 板框与安装尺寸建议

第一版不建议追求极限小型化。OWG 需要接线端子、备用电源、阀门驱动和测试点，板子太小会显著增加调试和安装难度。

推荐第一版优先适配 `Raspberry Pi 4B` 外壳生态。

建议尺寸：

| 版本 | 建议尺寸 | 适用场景 |
|---|---:|---|
| Pi 4B 外壳适配版 | 85 mm x 56 mm | 优先适配 Raspberry Pi 4B 外壳和安装柱 |
| 放宽调试版 | 100 mm x 80 mm | 仅用于桌面调试，不作为外壳适配目标 |

正式第一版优先按 `85 mm x 56 mm` 规划，外尺寸不要超过 `100 mm x 100 mm`。

原因：

- Pi 4B 外壳成熟、便宜、容易买到。
- 安装柱、外壳、透明壳、DIN 导轨转接件等生态现成。
- 项目早期不用把精力消耗在外壳设计上。
- `85 mm x 56 mm` 仍能放下 ESP32-S3、传感器输入、阀门驱动和必要测试点，但端子选择要更克制。

重要限制：

- Pi 4B 外壳的 USB、网口、HDMI、Type-C 开孔不一定适合 OWG 端子。
- 适配 Pi 4B 外壳主要是适配板框和固定柱，不等于所有接口天然对孔。
- 第一版应优先选择可开孔、可改侧板、透明上盖或开放式 Pi 4B 外壳。
- 阀门、电源、探头线缆最好集中从一侧或上盖出线，不要强行复用 Pi 4B 原有接口孔。

## 美国线盒兼容注意

美国常见住宅不是中国 `86 盒` 体系，而是 `1-gang`、`2-gang` 这类 gang box 体系。

常见参考尺寸：

- 标准美式 1-gang 面板约 `2.75 in x 4.5 in`，约 `70 mm x 114 mm`。
- 常见 1-gang 盒体开口量级约 `2 in x 3 in`，约 `51 mm x 76 mm`，深度随盒型变化。

这意味着：

- `100 mm x 80 mm` 主控板不适合塞进标准美式 1-gang 线盒。
- OWG 主控板应优先设计成阀门附近的独立表面安装控制盒。
- 不建议把阀门驱动、电池、传感器端子和备用电源都塞进墙壁线盒。
- 不要把 OWG 低压控制板和市电开关/插座混装在同一个线盒里，除非后续按相应电气规范重新设计隔离、认证和安装方式。

如果后续需要美式墙盒兼容，建议拆成两块板：

- 主控板：阀门附近独立安装，尺寸仍控制在 `100 mm x 100 mm` 内。
- 墙面 UI 子板：只放状态灯、按钮、小屏或蜂鸣器，目标尺寸约 `45 mm x 70 mm` 以内，适配 1-gang 面板空间。

第一版不以美式墙盒内置为目标。第一版优先保证阀门、电源、传感器和接线可靠。

固定孔建议：

- 优先兼容 Raspberry Pi 4B 安装孔位置。
- 螺丝规格按外壳确定，常见 Pi 外壳多使用 M2.5/M3 级别小螺丝。
- 孔径和孔位必须以实际外壳或官方机械图为准，打样前不要只凭记忆画孔。
- 固定孔周围留禁布区，避免螺丝压到走线或器件。

边缘预留建议：

- 接线端子距板边 2-4 mm，方便线缆从边缘或上盖进入。
- 大电流阀门端子和电源端子放在同一侧或相邻侧。
- 传感器端子集中放在另一侧，降低接错概率。
- USB/下载口不要被端子或外壳挡住。
- ESP32 天线朝外侧，天线区域不要靠近阀门线、电源线、大金属端子或外壳金属件。

外壳预留建议：

- PCB 外形尽量矩形，不要第一版就做异形板。
- 至少预留 10-15 mm 外壳高度给接线端子和线缆弯折。
- 如果使用螺丝端子，优先选择可开孔 Pi 4B 外壳，侧面或上盖要给螺丝刀留操作窗口。
- 状态灯、按钮和蜂鸣器位置要考虑外壳开孔。

尺寸不要在第一版定死成产品形态，但外形边界要控制在 `100 mm x 100 mm` 以内。第一版优先验证 Pi 4B 外壳能否容纳端子、线缆、备用电源和阀门驱动散热。

## Pi 4B 外壳适配布板建议

在 `85 mm x 56 mm` 约束下，端子和器件必须分层取舍。

建议：

- 使用低矮、可插拔端子，避免全部使用大号螺丝端子。
- 阀门和电源端子可保留较大间距，传感器端子可使用更紧凑的 JST/XH/PH 接口。
- 备用电池不一定放进同一个 Pi 4B 外壳，可以用外置电池盒或 UPS 模块。
- 蜂鸣器、状态灯、按钮尽量靠近上盖可开孔区域。
- 如果需要墙面按钮或显示，优先做独立 UI 子板，不要挤占主控板端子空间。
- 打样前先买目标 Pi 4B 外壳，实测内部高度、安装柱位置、侧面开孔和上盖空间。

## 漏水探头接口

第一版建议预留 3-4 路有线漏水探头接口。

每一路建议具备：

- 独立输入引脚。
- 上拉或下拉电阻。
- 简单 RC 滤波。
- ESD/浪涌保护预留。
- 接口旁清晰丝印，例如 `LEAK1_WASHER`、`LEAK2_HEATER`。
- 可选端子：PH2.0、XH2.54、螺丝端子，按安装场景取舍。

规则：

- 探头输入不应直接裸奔进 ESP32 GPIO。
- 长线传感器要考虑静电、误触发和潮湿环境下的漏电流。
- 每路探头最好能在固件里独立启用、禁用和命名。

## 水流传感器接口

水流传感器通常是脉冲输出，PCB 应考虑：

- 输入电平是否兼容 3.3V。
- 是否需要上拉电阻。
- 是否需要光耦、比较器或限流保护。
- 长线引入的噪声。
- 接口电源电压标注清楚，避免 5V/12V 接错。

至少预留一个测试点，方便示波器或逻辑分析仪观察脉冲。

## 阀门驱动接口

第一版推荐电动球阀，不推荐电磁阀。

PCB 需要明确：

- 阀门供电电压：12V 或 24V。
- 阀门动作方式：两线正反转、三线开关型、还是带限位反馈。
- 驱动器件：继电器、H 桥、MOSFET 或专用电机驱动。
- 反向电动势保护。
- 过流保护或保险丝。
- 阀门供电和主控供电的地线策略。

阀门接口丝印必须非常清楚，不要只写 `MOTOR`。建议写成：

- `VALVE_PWR`
- `VALVE_OPEN`
- `VALVE_CLOSE`
- `VALVE_GND`
- `VALVE_FB_OPEN`
- `VALVE_FB_CLOSE`

具体命名按阀门类型调整。

## 电源路径

漏水可能伴随跳闸，PCB 必须把备用电源当成安全相关设计。

建议支持：

- 主电源输入。
- 备用电池或 UPS 输入。
- 主电源存在检测。
- 备用电源电压检测。
- 低电量报警。
- 电源切换状态日志。
- 至少一次关阀动作所需的能量预算。

注意：

- 阀门动作电流可能远高于 ESP32 工作电流。
- 备用电源不只要带得动主控，还要评估能不能带动阀门。
- 降压模块要留足余量，避免阀门动作时主控复位。

## 实体按钮

建议至少有：

- `MUTE`：消音。
- `CLOSE`：手动关阀。
- `OPEN`：手动开阀。
- `MAINT`：维护模式，是否独立按键按外壳空间决定。

按钮输入建议：

- 硬件上拉或下拉。
- 简单 RC 防抖或固件防抖。
- 与高电流阀门走线保持距离。

## 状态灯和蜂鸣器

状态灯至少支持：

- 正常。
- 警告。
- 高风险。
- 已关阀。
- 维护模式。
- 电源异常。

蜂鸣器建议可由 MOSFET 控制，并保留关闭或限流设计，避免调试阶段过吵。

## 调试和测试点

第一版 PCB 一定要留测试点。

建议测试点：

- 3.3V。
- 5V 或主控输入电压。
- 12V/24V 阀门电源。
- GND。
- 每路漏水探头输入。
- 水流脉冲输入。
- 阀门开/关控制信号。
- 电池电压检测点。
- UART TX/RX。

没有测试点，后期排查会非常痛苦。

## 接口防呆和丝印

OWG 面向真实家庭安装，接口必须尽量减少接错。

建议：

- 所有端子旁写清电压和方向。
- 阀门、电源、传感器端子形态尽量不同。
- 高风险接口增加防反接或保险。
- 丝印使用英文为主，方便开源发布。
- 每个接口在文档和 PCB 上使用同一名称。

## 可制造性

第一版建议先追求稳定和可手工调试。

建议：

- 2 层板可以先做，但电源和阀门电流路径要保守。
- 阀门电流走线加宽。
- 电源入口附近留保险丝或自恢复保险丝位置。
- 接线端子边缘留足螺丝刀操作空间。
- 固定孔位置考虑外壳和墙面安装。
- ESP32 天线区域不要铺铜或靠近大金属件。

## 第一版 PCB 检查清单

画板前确认：

- 阀门型号和驱动方式已确定。
- 水流传感器输出电平已确定。
- 漏水探头线长和接口形式已确定。
- 主电源和备用电源方案已确定。
- 外壳尺寸和端子方向已有初步判断。

打样前确认：

- ESP32 下载、复位、BOOT 可用。
- 所有传感器输入有保护或限流。
- 阀门驱动有反灌和过流保护。
- 电源切换不会导致主控重启。
- 关键测试点齐全。
- 丝印没有模糊命名。

上电前确认：

- 先不接阀门，只测主控和传感器。
- 再接假负载测试阀门驱动。
- 最后接真实电动球阀。
- 第一次真实关阀测试必须有人在现场观察。
